# Semana 04 - Herencia

## Estructura del Proyecto

- `Content.java` - Clase padre con atributos protected
- `Movie.java` - Subclase para películas
- `Series.java` - Subclase para series
- `Documentary.java` - Subclase para documentales
- `Main.java` - Demostración de polimorfismo
- `JERARQUIA.md` - Documentación de la jerarquía

## Ejecución

```bash
javac *.java
java Main
```

## Características Implementadas

- Herencia con `extends`
- Atributos `protected` en clase padre
- Uso de `super()` en constructores
- Métodos sobrescritos con `@Override`
- Polimorfismo con arrays
- `instanceof` para identificar tipos
